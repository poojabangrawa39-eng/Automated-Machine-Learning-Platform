# 🤖 Automated Machine Learning Platform

## 📌 Project Overview

Automated Machine Learning Platform is an end-to-end Machine Learning solution that allows users to upload CSV or Excel datasets and automatically perform data preprocessing, feature selection, model training, evaluation, and prediction without requiring extensive coding knowledge.

The platform intelligently analyzes the uploaded dataset, identifies suitable target variables, applies appropriate Machine Learning algorithms, and generates predictions through an interactive Streamlit-based web interface.

## 🚀 Features

- Upload CSV or Excel datasets
- Automatic data preprocessing
- Missing value handling
- Categorical data encoding
- Automatic model training
- Supports Regression and Classification tasks
- Real-time prediction generation
- Interactive Streamlit dashboard
- User-friendly interface

## 📊 Functionalities

- Dataset Upload
- Data Cleaning
- Feature Engineering
- Model Selection
- Model Training
- Performance Evaluation
- Prediction Generation
- Result Visualization

## ⚙️ How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/Automated-Machine-Learning-Platform.git
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Run the application

```bash
streamlit run app.py
```

## 🎯 Project Outcome

The platform automates the complete Machine Learning workflow, enabling users to build predictive models from uploaded datasets with minimal manual effort. It simplifies data analysis, model building, and prediction generation, making Machine Learning accessible to beginners and non-technical users.
